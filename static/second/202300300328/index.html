游戏设置

游戏规则如下：
在这个游戏中，棋盘上共有n枚宝石。你将与电脑之神对战，分别轮流取走宝石。每次至少取l枚，至多取r枚，但不能不取（即 0 < l ≤ r ≤ n）。谁取到最后一枚宝石，谁就能获得宝藏，成为传说中的智者。作为玩家的你可以设置l、r、n的值并选择自己是先手还是后手。

不过，据说聪明的电脑之神会根据最优解出手？!可要小心咯。
